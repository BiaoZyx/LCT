# LCT - Line Cipher Translator

**线段密码翻译器**，支持文本与线段密码的双向转换，提供 **Nano 风格全屏界面** 和 **命令行一次性模式**。

## 功能亮点

- 双向翻译：文本 -> 线段密码，线段密码 -> 文本
- Nano 风格交互界面（基于 curses）：
  - 顶部标题栏、底部快捷键栏始终固定
  - 中间内容区可自由滚动，自动分隔每次输入
  - 彩色输出，支持中英文界面
- 命令行一次性模式：适合脚本调用
  ```bash
  lct -t "HELLO"          # 文本 -> LC
  lct -l "|-| ___ |- |- ---"  # LC -> 文本
  lct --dict              # 显示彩色字典
  lct --version           # 版本信息
  ```
- 完整字典：涵盖 A-Z、0-9 及常见标点符号（动态生成，修改 `LC_CODE_DICT` 自动更新）
- 智能配色：成功/错误/提示/分隔线均有独立颜色
- 中英文切换：交互模式手动选择，命令行自动检测系统语言
- 命令历史：上下箭头回溯，`/history` 查看最近 20 条
- 可打包为独立可执行文件（PyInstaller），无需 Python 环境

## 系统要求

- Python 3.8+
- 依赖库（均已纳入标准库或可通过 pip 安装）：
  ```bash
  pip install wcwidth windows-curses   # Windows 需要安装 windows-curses
  ```
  Linux/macOS 自带 `curses`，只需安装 `wcwidth`。

## 快速开始

### 从源码运行

```bash
git clone https://github.com/BiaoZyx/LCT
cd LCT
python LCT-v1.3.py
```

### 直接使用交互模式

```bash
python LCT-v1.3.py
# 选择语言后进入全屏操作界面
```

### 命令行一次性使用

```bash
python LCT-v1.3.py -t "HELLO"                    # 翻译文本
python LCT-v1.3.py -l "|-| ___ |- |- ---"       # 翻译线段密码
python LCT-v1.3.py --dict                        # 打印彩色字典
python LCT-v1.3.py --version                     # 显示版本
python LCT-v1.3.py --lang zh -t "示例"           # 强制中文 + 翻译
```

## 交互模式命令

进入交互界面后，底部快捷键栏会提示所有命令：

| 命令          | 功能               |
| ------------- | ------------------ |
| `1`           | 文本 -> 线段密码   |
| `2`           | 线段密码 -> 文本   |
| `3`           | 显示完整密码字典   |
| `/menu`       | 显示命令菜单       |
| `/help`       | 详细帮助           |
| `/version`    | 版本信息           |
| `/clear`      | 清空输出区         |
| `/history`    | 最近 20 条命令     |
| `/exit`       | 退出程序           |
| `/egg <颜色>` | 彩蛋 (red, green…) |

## 打包成可执行文件

使用 PyInstaller 生成独立程序（无需 Python 环境）：

```bash
pip install pyinstaller
pyinstaller --onefile LCT-v1.3.py
```

- 生成的二进制文件位于 `dist/` 目录。
- 你可以将其复制到系统路径（如 `/usr/local/bin/lct`）随时调用。
- **跨平台注意事项**：在 Windows 上打包前需安装 `windows-curses`，打包后的 `.exe` 即可在 Windows 运行。
